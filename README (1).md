## What is TalkTiles?

TalkTiles is an **Augmentative and Alternative Communication (AAC)** picture board that runs entirely in a web browser. Students, children, and adults who are non-verbal or have limited speech tap illustrated picture buttons to build sentences, which the app speaks aloud.

Commercial AAC apps cost $200–$300 and lock families into closed ecosystems. TalkTiles is:

- ✅ **100% free** — no subscription, no account, no ads, ever
- ✅ **Open source** — MIT licensed, fork it, improve it, share it
- ✅ **Works offline** — install once, use anywhere with no internet
- ✅ **Works on every device** — any browser on iPhone, iPad, Android, Chromebook, Windows, Mac
- ✅ **AI-powered** — generate custom vocabulary packs for any topic instantly
- ✅ **Community library** — browse and install ready-made packs in one tap
- ✅ **Branching navigation** — drill down from broad choices to specific ones
- ✅ **QR code sharing** — share vocabulary packs by displaying a QR code
- ✅ **Privacy-first** — no data ever leaves the device, no tracking, no cloud required

## Features

### 🖼️ Picture Communication Board
- 60+ hand-crafted illustrated picture buttons built in
- 6 starter categories: Feelings, Food & Drink, Fun & Activities, Animals, Places, People
- Always-visible **Core Word Strip** at the bottom — Yes, No, Want, Help, More, Stop, Go, Finish
- Large touch-friendly buttons designed for iPads and touchscreens

### 🌿 Branching Navigation — Drill-Down Choices
This is what makes TalkTiles feel like a real AAC system.

Instead of overwhelming a user with hundreds of buttons on one screen, branching lets you organize choices into layers. Tap a broad category, then drill into specific options — exactly the way people think and decide.

**Real example — Meal Planning:**
- Tap **🥩 Meat** → panel slides up showing: Meatballs, Hamburger, Steak, Tacos, Meatloaf
- Tap **Meatballs** → added to sentence, app speaks it, panel closes

**Built-in Meal Planning category includes:**

| Top-Level Choice | Sub-Choices |
|---|---|
| 🍗 Chicken | Roast Chicken, Chicken Soup, Chicken Pasta, Grilled, Tenders |
| 🥩 Meat | Meatballs, Hamburger, Steak, Tacos, Meatloaf |
| 🐟 Fish | Salmon, Tuna, Fish Sticks, Shrimp |
| 🥦 Vegetarian | Pasta, Pizza, Salad, Soup |
| 🥔 Side Dish | French Fries, Rice, Corn, Mashed Potatoes, Bread |
| 🍰 Dessert | Ice Cream, Cake, Cookies, Fruit |

Symbols with branches show a small **▶** indicator so the user knows more choices are coming. A breadcrumb trail at the top of every branch panel shows exactly where you are. Works to any depth — branches can have branches.

**Anyone can add branches to any symbol in Edit Mode** — no coding required.

### 💬 Sentence Builder with Text-to-Speech
- Tap pictures to build a sentence word by word
- Sentence bar shows all selected words as tappable chips — tap any chip to remove it
- **🔊 Speak** button reads the full sentence aloud using the device's built-in voice engine
- **Auto-speak mode** speaks each word the instant it is tapped
- Works in English by default — additional languages available via community vocabulary packs

### 🌐 Community Library — AI-Powered Pack Generator
Type any topic and get a fully built vocabulary pack in seconds.

**Featured Packs — ready to install in one tap:**
- 🍔 Ordering Food — restaurant, cafeteria, and community outing vocabulary
- 🔬 Science Lab Safety — equipment, procedures, and safety rules
- ☀️ Morning Routine — daily check-in, readiness, and self-advocacy
- 🏪 Community Outings — stores, libraries, parks, and public transit
- 📖 Book Club Feelings — opinions, characters, and literary discussion
- 💼 Job Interview Skills — transition and workplace communication

**AI Pack Generator:**
1. Type any topic — *"Ordering food," "Water cycle," "Feelings about starting high school," "Doctor's office visit"*
2. Select a grade level — PreK–K through Adult
3. Tap **✨ Generate** — AI builds 12–16 custom symbols tailored to your topic and grade
4. Tap **⬇ Install** — appears instantly as a new category on the board

### 📲 QR Code Sharing — Classroom Superpower
- Tap **📲 Share QR** to generate a QR code for your current vocabulary
- Choose **"This Category Only"** for a specific lesson or **"All Categories"** for the full board
- Students scan with their camera — vocabulary loads on their device in 5 seconds
- Project on your smartboard — the entire class is set up in under 30 seconds
- Share the link via Google Classroom, Remind, email, or text

### ✏️ Full Edit Mode — Open for Everyone
- Tap **✏️ Edit** to enter edit mode
- **Add any symbol** — upload a photo from your camera roll, paste an image URL, or choose an emoji
- **Add branches to any symbol** — turn any button into a drill-down menu with sub-choices
- **Delete symbols** with a single tap
- **Create new categories** with a custom name, emoji, and color
- All changes save automatically to the device

### ⬆⬇ Import / Export — Community Sharing
- Export your full vocabulary as a `.json` file to share with anyone
- Import a vocabulary pack from another teacher, SLP, or community member with one tap
- JSON format is open and human-readable — any developer can build tools around it

## How Teachers Use TalkTiles

### The QR Code Classroom Workflow

| Time | Action |
|---|---|
| **Night before** | Open Library → search your lesson topic → generate vocab pack → copy QR link |
| **Start of class** | Project QR code on smartboard |
| **Students** | Scan with camera → TalkTiles opens with lesson vocabulary already loaded |
| **During lesson** | All students — verbal and non-verbal — participate with the same tool |

### Subject-by-Subject Ideas

**Science** — Lab safety vocabulary, equipment names, experiment steps. Students tap picture buttons to label diagrams and describe processes.

**Social Studies** — People, places, events, and feelings vocabulary for historical topics. Students build sentences: *"George Washington — led — the army."*

**Language Arts** — Generate a pack tied to a specific book — characters, settings, emotions, plot events. Students retell the story by tapping in sequence.

**Health & SEL** — Morning check-ins where every student taps how they feel and why. Branching lets you go from *Feelings* → *Happy* → *Excited about something* → *My birthday*.

**Life Skills** — Meal planning at home, ordering at restaurants, shopping, job interviews. Branching navigation mirrors real decision-making: *"What do you want for dinner?"* → Chicken → Chicken Soup.

**Foreign Language** — Generate a pack where labels are in English but speech output is in the target language.

## For Developers

TalkTiles is a single HTML file — no build system, no dependencies, no framework. This is intentional: it maximizes accessibility, longevity, and ease of contribution.

### Tech Stack
- Vanilla HTML, CSS, JavaScript — no frameworks, no build step
- Inline SVG for all built-in picture symbols
- Web Speech API for text-to-speech
- localStorage for persisting vocabulary across sessions
- Service Worker for offline / PWA support
- QR code generation via qrcodejs (CDN)
- AI vocabulary generation via Anthropic Claude API

### Vocabulary JSON Format
Symbols support an optional `branch` array for drill-down navigation:

```json
{
  "categories": [
    {
      "id": "meal_planning",
      "label": "Meal Planning",
      "color": "#DC2626",
      "emoji": "🍽️",
      "symbols": [
        {
          "id": "meat",
          "label": "Meat",
          "speech": "Meat",
          "emoji": "🥩",
          "branch": [
            {
              "id": "meatballs",
              "label": "Meatballs",
              "speech": "I want meatballs please",
              "emoji": "🍝"
            },
            {
              "id": "hamburger",
              "label": "Hamburger",
              "speech": "Hamburger",
              "emoji": "🍔"
            },
            {
              "id": "steak",
              "label": "Steak",
              "speech": "Steak",
              "emoji": "🥩"
            }
          ]
        }
      ]
    }
  ]
}
```

Branches can be nested to any depth. A symbol with no `branch` property speaks immediately when tapped. A symbol with a `branch` array opens a drill-down panel instead.
