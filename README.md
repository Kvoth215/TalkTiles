Works on any device — iPhone, iPad, Android, Chromebook, PC — with no install required.

🔗 **Live App: [kvoth215.github.io/TalkTiles](https://kvoth215.github.io/TalkTiles/)**

## What is TalkTiles?

TalkTiles is an **Augmentative and Alternative Communication (AAC)** picture board that runs entirely in a web browser. Students, children, and adults who are non-verbal or have limited speech tap illustrated picture buttons to build sentences, which the app speaks aloud.

Commercial AAC apps cost $200–$300 and lock families into closed ecosystems. TalkTiles is:

- ✅ **100% free** — no subscription, no account, no ads, ever
- ✅ **Open source** — MIT licensed, fork it, improve it, share it
- ✅ **Works offline** — install once, use anywhere with no internet
- ✅ **Works on every device** — any browser on iPhone, iPad, Android, Chromebook, Windows, Mac
- ✅ **AI-powered** — generate custom vocabulary packs for any topic instantly
- ✅ **Community library** — 25 research-backed packs plus restaurant menus
- ✅ **Focus Mode** — present just 2–5 choices full screen for faster communication
- ✅ **Branching navigation** — drill down from broad choices to specific ones
- ✅ **QR code sharing** — share vocabulary packs instantly via QR code
- ✅ **Large, beautiful design** — built for touchscreens, easy to read at any distance
- ✅ **Privacy-first** — no data ever leaves the device, no tracking, no cloud required

## Features

### ⭐ Focus Mode — The Game Changer for Daily Life
Focus Mode clears the entire screen and shows only 2–5 large picture choices. This is the feature families use most at home and teachers use most in structured lessons.

**Research basis:** Light & McNaughton (2014) found that reduced choice sets significantly increase communication speed and accuracy for AAC users. Fewer options = faster, more confident responses.

**7 built-in scenes — launch with one tap:**

| Scene | Choices |
|---|---|
| 🛁 Bath or Shower | Bath · Shower — two giant half-screen buttons |
| 📺 TV Tonight | Cartoon · Movie · Sports · YouTube |
| 😴 Ready for Bed? | I am Tired · Not Yet · Read to Me |
| 🍽️ What for Dinner? | Chicken · Pizza · Pasta · Tacos |
| 🎮 Free Time | Video Games · Watch TV · Music · Outside · Read |
| 😊 How Do You Feel? | Happy · Sad · Angry · Tired · Okay |
| 👍 Yes or No | YES · NO — the simplest binary choice |

**Build your own custom scenes:**
- Name your scene (e.g. "Morning Choices" or "After School")
- Pick any symbols from any category on your board
- Choose 2–5 symbols
- Launch instantly or save permanently for next time
- Saved scenes appear in your Quick Scenes list forever

**What the play screen looks like:**
- 2 choices → two massive half-screen buttons
- 3 choices → three full-height columns
- 4 choices → 2×2 grid
- 5 choices → 2+3 layout
- Sentence bar at the top builds multi-word messages
- Speak, Clear, and Delete buttons always visible
- ✕ Exit returns to the full board instantly

### 🖼️ Picture Communication Board
- 60+ hand-crafted illustrated picture buttons built in
- 6 starter categories: Feelings, Food & Drink, Fun & Activities, Animals, Places, People
- Always-visible **Core Word Strip** at the bottom — Yes, No, Want, Help, More, Stop, Go, Finish
- Large, bold touch-friendly buttons designed for iPads and touchscreens
- Designed to be readable from across a classroom or at arm's length

### 🌿 Branching Navigation — Drill-Down Choices
Instead of overwhelming a user with hundreds of buttons on one screen, branching organizes choices into layers. Tap a broad category, then drill into specific options.

**Built-in Meal Planning category:**

| Top-Level | Sub-Choices |
|---|---|
| 🍗 Chicken | Roast Chicken, Chicken Soup, Chicken Pasta, Grilled, Tenders |
| 🥩 Meat | Meatballs, Hamburger, Steak, Tacos, Meatloaf |
| 🐟 Fish | Salmon, Tuna, Fish Sticks, Shrimp |
| 🥦 Vegetarian | Pasta, Pizza, Salad, Soup |
| 🥔 Side Dish | French Fries, Rice, Corn, Mashed Potatoes, Bread |
| 🍰 Dessert | Ice Cream, Cake, Cookies, Fruit |

Symbols with branches show a **▶** indicator. A breadcrumb trail shows where you are. Works to any depth. Anyone can add branches to any symbol in Edit Mode.

### 💬 Sentence Builder with Text-to-Speech
- Tap pictures to build a sentence word by word
- Sentence bar shows all words as tappable chips — tap to remove
- **🔊 Speak** reads the full sentence aloud using the device's built-in voice engine
- **Auto-speak mode** speaks each word the instant it is tapped

## How Teachers Use TalkTiles

### The QR Code Classroom Workflow

| Time | Action |
|---|---|
| Night before | Open Library → find or generate lesson vocab → copy QR link |
| Start of class | Project QR code on smartboard |
| Students | Scan with camera → TalkTiles opens with lesson vocabulary loaded |
| During lesson | All students — verbal and non-verbal — participate with the same tool |

### The Focus Mode Classroom Workflow

| Situation | What to do |
|---|---|
| Binary question ("Do you want X or Y?") | Open Focus Mode → Yes or No scene |
| Lesson choice ("Which topic interests you?") | Build a 3-4 choice custom scene |
| Morning check-in | How Do You Feel? scene |
| Choice board replacement | Build a custom scene with today's options |

### Subject-by-Subject Ideas
**Science** — Library has Water Cycle, Solar System, Human Body, and Scientific Method packs ready to install.

**Social Studies** — American Revolution, Civil War, and US Government packs built from actual curriculum vocabulary.

**Life Skills** — Take the class to a Red Robin or McDonald's? Launch the restaurant menu, let students order themselves.

**SEL** — Morning check-in with the How Do You Feel? Focus scene. Every student participates.

**Transition** — Job Interview Skills and Workplace Communication packs support IEP transition goals.

## For Developers

TalkTiles is a single HTML file — no build system, no dependencies, no framework.

### Tech Stack
- Vanilla HTML, CSS, JavaScript — no frameworks, no build step
- Inline SVG for all built-in picture symbols
- Web Speech API for text-to-speech
- localStorage for persisting vocabulary and Focus Mode scenes
- Service Worker for offline / PWA support
- QR code generation via qrcodejs
- Optional AI vocabulary generation via Anthropic Claude API

### Vocabulary JSON Format with Branching

```json
{
  "categories": [
    {
      "id": "meal_planning",
      "label": "Meal Planning",
      "color": "#DC2626",
      "emoji": "🍽️",
      "symbols": [
        {
          "id": "meat",
          "label": "Meat",
          "speech": "Meat",
          "emoji": "🥩",
          "branch": [
            { "id": "meatballs", "label": "Meatballs", "speech": "I want meatballs please", "emoji": "🍝" },
            { "id": "hamburger", "label": "Hamburger", "speech": "Hamburger", "emoji": "🍔" },
            { "id": "steak", "label": "Steak", "speech": "Steak", "emoji": "🥩" }
          ]
        }
      ]
    }
  ]
}
```
