## What is TalkTiles?

TalkTiles is an **Augmentative and Alternative Communication (AAC)** picture board that runs entirely in a web browser. Students, children, and adults who are non-verbal or have limited speech tap illustrated picture buttons to build sentences, which the app speaks aloud.

Commercial AAC apps cost $200–$300 and lock families into closed ecosystems. TalkTiles is:

- ✅ **100% free** — no subscription, no account, no ads, ever
- ✅ **Open source** — MIT licensed, fork it, improve it, share it
- ✅ **Works offline** — install once, use anywhere with no internet
- ✅ **Works on every device** — any browser on iPhone, iPad, Android, Chromebook, Windows, Mac
- ✅ **Privacy-first** — no data ever leaves the device, no tracking, no cloud

**[kvoth215.github.io/TalkTiles](https://kvoth215.github.io/TalkTiles/)**

## Features

### 🖼️ Picture Communication Board
- 25+ built-in illustrated tiles across 5 starter categories
- Categories: **Daily Life, Social & SEL, Academic, School, Transitions**
- Always-visible **Core Word Strip** (I, want, need, yes, no, help, more, stop, go, please, thank you…)
- Small, Medium, and Large tile sizes for different vision and motor needs

### 💬 Sentence Builder with Text-to-Speech
- Tap tiles to build a sentence word by word
- **🔊 Speak** button reads the full sentence aloud
- **Word-by-word highlighting** — each word chip glows yellow as it's spoken, helping students follow along
- **Hold-to-speak** — press and hold any tile to preview its word; release to stop (reduces accidental activations)
- **Auto-speak mode** — speaks each word the instant it's tapped

### ↩️ Undo / Redo
- Full undo and redo history for the sentence bar
- Buttons go grey when there's nothing left to undo or redo
- Critical for AAC users who tap accidentally or want to revise

### 📝 Sentence Templates
- One-tap starters: **I want… / I feel… / I need… / I am… / I don't want… / Please help / Can I… / I like…**
- Loads the opening words instantly so the student just picks what comes next
- Reduces cognitive load and speeds up communication

### 💡 Smart Suggestions
- After each tap, a **Next word** strip appears with up to 6 suggested tiles
- Suggestions are ranked by each student's own usage history — the more they use a tile, the higher it ranks
- Unique to each profile, so every student gets personalized predictions

### 👤 Multi-Profile System
- Create unlimited student profiles — each stores its own vocabulary, categories, and usage data
- Switch profiles instantly from the top bar
- Each profile is completely independent on the same device

### 🔏 Kiosk / Lock Mode
- Lock the interface so students only see the communication board
- All teacher controls, settings, and edit tools are hidden
- One-tap unlock for staff

### ✏️ Full Vocabulary Editor
- Add, edit, or delete any tile
- Choose from **ARASAAC pictogram library** (searchable, 30,000+ symbols), **emoji**, **uploaded photos**, or **image URLs**
- Create and color-code custom categories
- Assign any word as the spoken text, independent of the label

### 📊 Usage Analytics
- Tracks how often each tile is used, per student profile
- Visual bar chart of the top 15 most-used symbols
- Export data for SLP or IEP documentation

### 📲 QR Code Sharing
- Generate a QR code to share any vocabulary pack instantly
- Students scan to load the vocabulary on any device in seconds
- Share a single category or the full board

### 🌐 Community Library
- Built-in vocabulary packs ready to install in one tap:
  - **Extended Emotions Pack** — 12 feeling tiles for SEL
  - **Food & Snack Requests** — cafeteria and snack choices
  - **Classroom Communication** — pencil, paper, my turn, repeat, I know

### ⌨️ Switch Scanning
- Full keyboard scanning support for motor-impaired users
- **Space** = move to next item · **Enter** = select · **Esc** = stop scanning
- Works across both the core strip and the symbol grid

### 🖨️ Print Mode
- Print the current vocabulary board as a paper backup
- Clean layout, no UI chrome

### 📥 Export / Import
- Export any student's full vocabulary as a `.json` file
- Import to restore or move to another device

## Classroom Setup Tips

**One iPad, one student**
Lock the device into a specific profile using Lock Mode. The student sees only the board.

**Shared classroom device**
Create a profile for each student. Switch profiles from the top bar dropdown. Each student's vocabulary and usage data stays separate.

**Sending boards home**
Use the QR Share button. Parents scan with any phone camera — no app, no account needed.

**Documenting for IEPs**
Open Usage Stats (📊) to see a frequency chart of each student's most-used words. Screenshot or note these for IEP documentation or SLP sessions.
